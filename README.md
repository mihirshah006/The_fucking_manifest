How to Build?
-------------

To initialize your local repository using the PolyOS sources, use a 
command like this:

```bash
  repo init -u git://github.com/PolyOS-CAF/android.git -b nougat
```
  
Then to sync up:
----------------

```bash
  repo sync -c -jx --force-sync --no-clone-bundle --no-tags
```
